# Wave App 🌊
React、Open-Meteo API、Leafletを使用して作成した波情報ダッシュボードです。

## 構成図
(ここに構成図の画像をドラッグ＆ドロップすると、GitHub上で表示されます)

## 機能
- 地図をクリックして詳細な波情報を取得
- 小数点第4位までの精密な座標表示
- 波高データのグラフ表示（Chart.js）

## 開発環境
- Frontend: React
- Backend: Node.js
- Infrastructure: AWS (今後デプロイ予定)
